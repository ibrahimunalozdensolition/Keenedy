# Kennedy

Yönlendirme blog sitesi - Next.js ve Firebase ile geliştirilmiştir.

## Özellikler

- Blog yapısı (başlık, fotoğraf, açıklama)
- Yönlendirme sistemi
- Reklam yönetimi (Firebase'den manuel + AdSense desteği)
- Responsive tasarım
- Firebase Firestore entegrasyonu

## Teknolojiler

- Next.js 16
- TypeScript
- Tailwind CSS
- Firebase (Firestore)

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

## Build

```bash
npm run build
npm start
```
